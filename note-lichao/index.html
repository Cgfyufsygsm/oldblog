前言

因为最近在学斜率优化，然后遇到了 CEOI 的那道毒瘤题，被迫来学习李超树。

李超树是一种能维护平面内线段的最大值的数据结构。

概述

李超树的精髓在于维护每个区间内有可能成为最优解的线段。

定义一个线段在 处最优当且仅当其与 的交点纵坐标最大。

我们在区间 维护的最优线段满足如下性质：

  • 该线段完整覆盖
  • 该线段在区间中点最优

因此李超树就是一棵维护上面这样的信息的线段树。

BZOJ#3938. Robot

题意

个机器人，一开始分别在 位置。接下来进行 次操作：

  • ，表示在时间 时重置 号机器人的速度为 单位长度每秒， 可以为负。
  • ，表示查询时间 时距离原点最远的机器人的距离。

，总操作次数不超过 。

思路

分析可得，每个机器人的 图像都是分段的一次函数，所以可以离线，一开始建出一棵李超树来维护这些分段的一次函数的最值，再统一进行查询。（最大值就直接做，最小值为了方便可以把所有直线的 和 取相反数之后再做一次）。由于时间过大，所以需要离散化。

总体来说还是比较不好写的，需要注意的细节非常多。需要注意的是离散化之后计算中点的函数值时一定要取离散化后的中点，因为递归下去的时候是以离散化之后的中点为参照的，不然 WA 80。

同时，如果你同时维护了两棵李超树分别维护最大和最小值，修改函数千万要写成两个，否则在更新最大值的时候会更新很多无谓的最小值，会 T 飞掉只剩 40pts。

把思路理清楚就可以写代码了。
