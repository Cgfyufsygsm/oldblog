ABC202F - Integer Convex Hull

YangTY
OIsol

发布于：2021年5月27日
次浏览

Description

平面内 个坐标均为整数的点，其中任意三点不共线，令所有点 构成一个全集 ，子集 且 。问凸包面积为整数的子集 的个数模 的值。

。

Solution

比较神奇的题目。

考虑 Andrew 算法的执行流程：将所有点按照先 轴升序后 轴升序排序，然后从左往右弄出下凸包，再从右往左弄出上凸包。本题可以利用类似的思路。

然后我们由叉积计算面积的方法知道，整点三角形的面积乘二之后必然是整数。于是我们考虑依次用三角形弄出凸包，具体地，可以 dp：

  • 枚举凸包最左端的端点 。

  • ：凸包满足如下条件的点集 的数量：

    • 从 顺时针考虑每个点，最后两个分别是 和 ；
    • 所有的点都在直线 以上；
    • 面积乘以二之后模二为 。

  • ：凸包满足如下条件的点集 的数量：

    • 从 逆时针考虑每个点，最后两个分别是 和 ；
    • 所有的点都在直线 以下；
    • 面积乘以二之后模二为 。

  • 确定每个最右端的点 ，然后枚举 和 统计答案：

这样做的原理其实就是合并上下凸壳：根据上面设计的 dp 状态可知我们确定了左端点 和右端点 后，满足上凸壳的点集个数即为枚举倒数第二个点 然后求和的结果 ，满足下凸壳的点集个数即为 。

怎么计算 和 呢？我们可以通过 递推出 ，其原理就是判断 能否被加入凸壳中，能则更新，不能就更新 （加入下凸壳）。新加入的部分即为 ，需要根据这个面积的二倍的奇偶性判断更新哪个 。

这个做法要求我们提前预处理每个三角形的面积的二倍的奇偶性 和三角形内点的个数 。总时间复杂度 。
