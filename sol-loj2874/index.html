YangTY

Description

定义一段区间的加权众数为一个数字乘上其出现次数的最大值。

次询问区间加权众数。

Solution

莫队可以做区间众数我们是知道的。

但是这个题的众数带权，莫队的删除操作非常不好实现。这个时候就需要一个叫回滚莫队的科技，使得我们没有删除操作：

  • 将每个询问按以左端点所在块为第一关键字，右端点所在块为第二关键字排序。
  • 按顺序处理询问：
    • 如果左端点的块 与上一个询问的左端点的块不同，那么将莫队左端点初始化为 的右端点加 ，右端点初始化为 的右端点。
    • 如果询问的左右端点在同一块内， 暴力。
    • 如果询问的左右端点所在块不同：
      • 如果询问的右端点大于莫队区间的右端点，扩展莫队的右端点。
      • 扩展莫队的左端点。
      • 回答询问。
      • 撤销左端点的改动，回滚到 的右端点加 。

分析一下复杂度：设左右端点在一个块内有 次，不在一个块内有 次。

  • 不难得到回答第一类询问的复杂度为 。
  • 考虑对于左端点在一个块内的一类询问，假设有 次：
    • 莫队右端点扩展 次。
    • 左端点每次扩展 次。
    • 故一类询问的复杂度为 。
  • 所以总复杂度为 。

这告诉我们，回滚莫队的时间复杂度是对的。
