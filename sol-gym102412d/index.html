GYM102412D The Jump from Height of Self-importance to Height of IQ Level

YangTY
OI题解

发布于：2021年10月26日
次浏览

Description

长度为 的序列（）， 次操作，每次操作选定 并向右 shift 位。每次操作后回答排列中是否存在长度为 的上升子序列。

Solution

shift 这种操作可以抽象为将一段区间取出来然后塞到另外两段区间的中间，所以考虑使用平衡树来维护这个序列。

最关键的事情就变为了维护区间的信息。考虑我们维护一个区间里面是否存在需要的三元组。为了从左右儿子合并得到当前的信息，我们需要知道：

  • 左/右儿子中是否已经存在该三元组；
  • 该三元组从左儿子中抽两个加上右儿子中的一个组成；
  • 该三元组从右儿子中抽两个加上左儿子中的一个组成。

所以我们需要维护 （）中，最小的 ，这样更容易在右儿子中找到比 大的 。同理维护一个最大的 。这样的话，如果右儿子的最大值 大于左儿子的 ；或者左儿子的最小值 小于右儿子的 ，都视为找到合法的解。

答案，最小值，最大值都很好维护，现在主要矛盾是如何维护 和 。首先显然可以从子节点继承过来，然后考虑 和 分别在左右子树的情况，以维护 为例。一个很强的 是将最小值作为 然后在右子树中递归找 的后继。
