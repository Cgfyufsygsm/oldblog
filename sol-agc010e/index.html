AGC010E - Rearranging

YangTY
OIsol

发布于：2022年3月15日
次浏览

Description

题意：给定 个整数 。先手将其任意打乱，后手可以任意次交换任意两个相邻的互质的数。先手想最小化字典序，后手想最大化字典序，双方绝顶聪明，问最后结果。

Solution

关键性质：先手打乱数列之后，任意两个不互质的数对之间的相对位置都不会发生改变。那么我们不妨将所有不互质的数对连边，然后先手做的事情就是给其定向（显然会形成个 DAG），而后手做的事情就是找到这个 DAG 的最大字典序的拓扑序。

为了让字典序尽可能小，我们先将 排序，然后贪心地 dfs 给边定向。然后再随便找找其最大拓扑序即可。
